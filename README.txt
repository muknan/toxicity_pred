The toxicity prediction challenge (Kaggle) predicts the chemicals listed in the CSV which of them are toxic. 

Pre-requisites:

Anaconda - The distribution includes data-science packages (Python 3) suitable for Windows, Linux, and macOS.

Contents of final_ml.zip:
1. Final report_MLXMen.pdf
2. the-toxicity-prediction-challenge.zip
3. final.ipynb
4. README.txt
5. final-code.py
6. installpk.py


After Anaconda installation:
1. Extract the files from final_ml.zip to anaconda root directory.
2. Extract the dataset from the-toxicity-prediction-challenge.zip to anaconda root directory.
2. Open "Anaconda Prompt" from Start Menu or Dock.
3. Type "python installpk.py" in Anaconda Prompt to install the required packages.
4. Follow the prompts if any for successful installation.


For running the code:
1. Open "Anaconda Prompt".
2. Type "jupyter notebook" and press enter to open notebook server.
3. Press "Upload" button on top right and select "final.ipynb" from extracted files.
4. Click on newly added file by name "final.ipynb", that will open the notebook where you can execute the code.
